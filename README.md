# AI-Research-Agent-using-Autogen
It uses Microsoft’s AutoGen framework to orchestrate collaborative AI agents that scrape academic data from ArXiv and Google Scholar. The system evaluates five papers, extracts key pros and cons, and displays a summary via a Streamlit web UI. Built with Python, pyautogen, and scholarly.

```
pip install -r requirements.txt
```

For running this project use
'''
"python -m streamlit run app.py"
'''
